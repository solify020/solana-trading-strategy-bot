# Solana Meteora Trading Bot (TypeScript scaffold)

This is a minimal TypeScript + Node scaffold created to bootstrap development.

Quick start

```powershell
npm install
npm run dev   # run in dev (ts-node-dev)
npm run build # compile to dist/
npm start     # run compiled code
npm test      # run unit tests (vitest)
```

What you get

- TypeScript configuration
- ESLint + Prettier config
- Vitest test runner with a sample test
- Basic src/ with a sample function

Next steps

- Add your trading bot implementation in `src/`
- Wire secrets using environment variables (do NOT commit them)
